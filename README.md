# AlarmClone

An app that clone original iOS system alarm, which is in the clock app.

## Contents

- Tab Bar Controller
- Navigation Controller
- Table View Controller
- UserDefaults
- Local Notification Push
- Pass data with protocol and delegate method
- Struct, Class and Enum
